# 🚀 Star Trek

A simple 2D space shooter built with **Python** and **Pygame**.

Fly your spaceship, shoot down enemy ships, and survive for as long as possible while trying to achieve the highest score.

## Features

- Player movement
- Enemy spawning
- Laser shooting
- Collision detection
- Score tracking
- Game over screen
- Sound effects
- Background music

## Controls

- **W** - Move Up
- **A** - Move Left
- **S** - Move Down
- **D** - Move Right
- **Space** - Shoot

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Tumelo-Meso/Space-Trek.git
```

2. Install Pygame:

```bash
pip install pygame
```

3. Run the game:

```bash
python main.py
```
